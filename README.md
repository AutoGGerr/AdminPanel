**Admin Panel**
Редактирование контента свёрстанного на **HTML/SCSS** лендинга с помощью админ панели, представленной как отдельная страница, написанной на чистом JS с применением NoSQL базы данных(FireBase/Firestore).
Запускать необходимо при помощи сборщика(Vite), тк применяется ассинхронное программирование.
Функционал админ панели:
    1.Вход осуществляется при помощи нажатии на кнопку "Войти" в самом лендинге(index.html).
    2.Редактирование всего текстового контента(за исключением footer/подвала сайта) в отдельных полях ввода.
    3.Сохранение через нажатие на кнопку "Сохранить".
    4.Возвращение на лендинг при нажатии на кнопку "На сайт".
